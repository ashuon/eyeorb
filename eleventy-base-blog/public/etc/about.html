welcome

this is a web clique dedicated to resonating with objects/concepts of a surreal/otherworldly nature, similar to the concept of "dreamcore" but please do not call it that..

what is a web clique?

a web clique, or simply just called clique, is a group of people with personal websites who all share a common interest or thing, and they put a piece of code on their website to signify this.

what are the rules?

  • you must have your own personal hand-coded website to put it on (this does not include carrds/rentry, social media sites, or messaging apps)
  • you must not have anything violently hateful or offensive on your site (i cant control people, so only abide by this if you were looking to be added to the members list)
  • please put the code up in an easily-findable place, if you want to be added to the list (this just makes things easier for me)
  • you can put in a custom object/concept if you want, just really make sure it fits the theme. you cannot put "cat" for example, because that is a super common thing and not immediately associated with anything surreal.
